## 🚀 What this app does

This app helps fix OpenCode sign-in issues with Claude from Anthropic. It is made for users who see an `invalid token` error and need to restore access.

Use it when:
- OpenCode cannot connect to Claude
- You keep getting sign-in or token errors
- You want a simple way to get back into your account on Windows

After install, use the app to restore your Claude access for OpenCode.

1. Start the app on your Windows PC.
